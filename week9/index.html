Notes


  1. The window.alert() method will pause the execution of the program and display a message in a dialog box. The message is provided as an argument to the method, and undefined is always returned
  2. The window.confirm() method will stop the execution of the program and display a confirmation dialog that shows the message provided as an argument, and give the options of OK or Cancel
  3. The window.location property is an object that contains information about the URL of the current page
  4. The host property returns a string describing the domain of the current URL and the port number (this is often omitted if the default port 80 is used):
    << window.location.host
  5. The pathname property returns a string of the path that follows the domain
  6. The window.history.go() method can be used to go to a specific page, where 0 is the current page
  7. The window.screen object contains information about the screen the browser is displayed on. We can find out the height and width of the screen in pixels using the height and width properties respectively
  8. To create a cookie, we assign it to JavaScript’s 'cookie jar', using the document.cookie property
  9. To see the current contents of the cookie jar, we can simply enter document.cookie
  10. Adding the string '; secure' to the end of a cookie will ensure it’s only transmitted over a secure HTTPS network
  11. To remove a cookie, we need to set it to expire at a time in the past
  12. If a browser supports the Web Storage API, the window object will have a property called localStorage, which is a native object with a number of properties and methods used to store data
  13. To remove an entry from local storage, use the removeItem method. Alternatively, this can be done using the delete operator
  14. To completely remove everything stored in local storage, use the clear() method
